Task: blocco di codice eseguibile in concorrenza con altro codice
    funzione
    functor
    lambda
Thread: singola esecuzione di un task in un processo

Ogni programma C++ ha almeno un thread, quello che esegue main()

Da C++11 la classe std::thread permette di gestire l'esecuzione un task
In alternativa è possibile usare boost::thread e classi connesse

Prima della terminazione del programma di solito si aspetta la terminazione dei thread creati via join()
- In alternativa, detach() lo lascia in esecuzione

Se una risorsa è acceduta da più thread dovrebbe avere un accesso regolamentato

Passaggio di argomenti a un task
- Nel caso di funzioni si usa un costruttore variadico template, è richiesto l'uso di ref() / cref() per le reference
